# Eclipse auto-complete

1. Eclipse -> Preferences -> Java -> Editor -> Content Assist 
2. `Auto activation triggers for Java:` you will find `.`
3. Use `abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ.` to replace `.`
4. apply
